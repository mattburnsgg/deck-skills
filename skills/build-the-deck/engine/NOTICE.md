# Vendored engine: frontend-slides

The files in this `engine/` folder are vendored from **frontend-slides** by zarazhangrui, used under the MIT License (see `LICENSE` in this folder).

- Source: https://github.com/zarazhangrui/frontend-slides
- Version: 1.0.0
- License: MIT

**Why vendored:** so a single install of `deck-skills` ships the full engine (the viewport-fitting base, 12 themes, animation patterns, PowerPoint import, PDF export) with no second download. To pull upstream updates later, replace these files from the source repo.

`frontend-slides-guide.md` is the engine's own SKILL.md, renamed so it is not loaded as a separate skill. `UPSTREAM-README.md` is the engine's original README, kept for reference.
